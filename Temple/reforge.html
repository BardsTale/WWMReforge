재련 제어

"재련" 버튼 클릭: 1. 활성화되어 있고 잠기지 않은 슬롯의 등급을 재련합니다. 2. 비활성 슬롯이 있으면 다음 슬롯 활성화를 시도합니다.
확률 정보:
• 활성화: 각 회당 3.33% 진행도 증가, 발생 가능: 즉시 활성화
• 재련 등급 확률:파랑82% / 보라15% / 금색3%
• 금색 천장: 90회재련 내 반드시 등장금색, 천장 후 카운트 초기화
• 5번 슬롯 활성화 후 반드시 금색 등급, 재련에 참여하지 않음
• 1번 슬롯은 다양한 속성 분기가 있고, 2~4번 슬롯은 세트 속성 분기가 있음
※ 활성화 진행도 100% 도달 시, 즉시 활성화현재 슬롯하고 자동으로 1회 재련

프리셋 관리 (최대 5개)
저장된 프리셋이 없습니다. 금색 등급이 나오면 프리셋을 저장할 수 있습니다.

사용 설명

  • 5번 슬롯활성화 후 반드시 금색이며, 속성은 고정: 금·요일
  • 초기 상태: 1번 슬롯은 활성화(파랑), 2~4번 슬롯은 비활성
  • 재련 기능은 두 작업을 동시에 수행합니다:1. 활성화되어 있고 잠기지 않은 슬롯 등급 재련;2. 다음 슬롯 활성화 시도
  • 등급 재련 확률:파랑82%, 보라15%, 금색3%
  • 금색 천장:각 슬롯독립 카운트, 90회재련 내 반드시 등장금색
  • 슬롯 잠금:체크슬롯 앞 체크박스로 잠금해당 슬롯, 재련 시잠금슬롯변경되지 않음
  • 중요: 1~4번 슬롯에서 금색 등급이 나오면 확인창이 뜨며 "프리셋 저장"을 선택할 수 있음
  • 프리셋 저장: 최대 5개까지 저장 가능하며, 활성화 진행도/등급/속성을 포함하고 금색 천장 카운트는 0으로 초기화됨
  • 프리셋 적용: 적용 시 프리셋에서 활성화된 슬롯은 현재 슬롯의 등급/속성을 덮어쓰고, 비활성 슬롯은 현재 상태(활성화 진행도 포함)를 유지함
  • 사용자 설정: "사용자 설정" 버튼으로 활성화 진행도, 등급, 속성 등 슬롯의 모든 상태를 직접 조정할 수 있음
  • 5번 슬롯 활성화 후 자동 잠금(변경 불가), 참여하지 않음: 재련
  • 이 도구는 기록용: 재련 과정, 비용 낭비 위험을 줄이는 데 도움
  • 모든 데이터는 브라우저에 자동 저장되며, 새로고침해도 사라지지 않습니다
